# AIPath 中文输入模板

把下面内容复制给 Agent，按自己的情况填写即可。不完整也可以先填，AIPath 会基于假设继续诊断。

```text
请使用 AIPath 帮我做一次 AI 转型路径诊断。

1. 目标岗位或 JD：
（粘贴目标岗位 JD；如果没有 JD，就写目标方向，比如 AI Engineer / FDE / AI PM / AI Ops / automation builder）

2. 当前背景：
（比如 SDE / Backend / Data / Product / Ops / Analyst / Student；工作年限；主要经历）

3. 工作经历或 workflow 例子：
（列 3-5 条你真实做过的事情）

4. 项目描述或 AI workflow 想法：
（你做过的 RAG / Agent / 自动化 / 数据分析 / AI 工作流项目；没有也可以写想做什么）

5. 当前 AI/LLM stack：
（比如 OpenAI API、Claude、LangChain、RAG、向量数据库、prompt、Cursor、Claude Code 等）

6. 部署或 production 证据：
（有没有上线、内部使用、demo、用户、录屏、README、部署链接）

7. 评估或指标证据：
（有没有 eval set、测试样例、准确率、人工 review、失败案例、latency/cost）

8. 目标公司类型：
（AI startup / B2B SaaS / 大厂 / 咨询 / 传统行业 AI 转型 / 不确定）

9. 时间线：
（比如 2 周、1 个月、3 个月、半年）

10. 最大担心点：
（比如简历不像 AI 岗、项目太浅、不是工程背景、不知道投什么岗位、怕面试被追问）

输出要求：
先给我 short diagnosis，不超过 500 字；如果方向靠谱，再展开完整报告。
```
